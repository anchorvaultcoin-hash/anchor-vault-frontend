# AnchorVaultV45 — Frontend & Landing

This repository contains the **landing page** and **web interface** for AnchorVaultV45 — a non-custodial, multi-asset ERC-20 vault with EIP-712 two‑factor authorization.

> **Smart contract source code is not public** (it is under BUSL-1.1 in a private repository). This repo only holds the frontend and documentation.

## Features

- **Landing page** – project overview, roadmap, links.
- **Web wallet interface** – connect wallet, create vault, deposit/withdraw, manage keys.
- **Testnet version** – currently live on Sepolia.

## Live demo

- GitHub Pages: `https://anchorvaultcoin-hash.github.io/anchor-vault-frontend` (replace with actual after deploy)
- Custom domain: `https://anchorvault.site` (once set up)

## Tech stack

- HTML5 / CSS3 / JavaScript (ES6)
- ethers.js v6 / Web3
- EIP-712 signature handling
